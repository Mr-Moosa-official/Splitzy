# Splitzy

A smart expense splitting app designed for groups to easily track and settle shared costs.

This is a Next.js app built with Firebase Studio.

## Core Features

- **Add Expense**: Allows users to add new expenses, specifying the amount, description, and involved group members.
- **Custom Split**: Offers options to split expenses unevenly, specifying custom amounts for each group member.
- **Balance Tracking**: Displays individual balances for each group member, showing who owes whom.
- **Debt Settlement**: Facilitates debt settlement by recording payments and updating balances accordingly.
- **Manage Groups**: Enables users to switch between multiple groups for different shared expenses.
- **Intelligent Suggestion Tool**: Suggests the optimal way to pay off all group debts by determining the minimum number of transactions and the respective amounts to be paid.
