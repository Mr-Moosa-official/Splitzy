# **App Name**: Splitzy

## Core Features:

- Add Expense: Allows users to add new expenses, specifying the amount, description, and involved group members.
- Custom Split: Offers options to split expenses unevenly, specifying custom amounts for each group member.
- Balance Tracking: Displays individual balances for each group member, showing who owes whom.
- Debt Settlement: Facilitates debt settlement by recording payments and updating balances accordingly.
- Manage Groups: Enables users to create and manage multiple groups for different shared expenses.
- Intelligent Suggestion Tool: Suggests the optimal way to pay off all group debts by determining the minimum number of transactions and the respective amounts to be paid. This tool enhances efficiency in settling balances among group members, minimizing complexity and streamlining the payment process.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to convey a sense of sophistication and trustworthiness.
- Background color: Dark charcoal (#212121) to provide a sleek and modern backdrop, enhancing the visibility of content.
- Accent color: Lavender (#BB86FC) to highlight key actions and elements, adding a touch of elegance and guiding user interaction.
- Body and headline font: 'Inter', a grotesque-style sans-serif providing a modern, neutral, and clean aesthetic suitable for both headlines and body text.
- Use minimalist, geometric icons in lavender (#BB86FC) to represent expense categories and actions.
- Employ a card-based layout for expenses and balances to maintain a clean and organized interface.
- Incorporate subtle animations during state changes, like adding an expense or settling a debt, to provide visual feedback and a sense of responsiveness.
